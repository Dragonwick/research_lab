# Ot2 Color Lab

OT-2 color mixing lab

Generated by MADSci template: `lab/minimal` v1.0.0

## Overview

This is a minimal MADSci lab configuration that runs without Docker.
It's ideal for getting started, development, and testing.

## Quick Start

### 1. Install MADSci

```bash
pip install madsci-client madsci-squid
```

### 2. Start the Lab

```bash
# Start the lab manager
python start_lab.py

# Or use the CLI
madsci start lab --mode local
```

### 3. Open the Dashboard

Navigate to http://localhost:8000 in your browser.

## Configuration

Default settings live in `settings.yaml` (safe to commit).
Secrets and environment-specific overrides go in `.env` (gitignored).
Environment variables override both files.

## Adding Nodes

To add a node to your lab:

1. Create a new module: `madsci new module --name my_device`
2. Add the node URL to your workcell configuration
3. Restart the lab
